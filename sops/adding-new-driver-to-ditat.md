# Adding New Driver to Ditat

## Purpose

To establish a standardized process for properly setting up a new driver in DITAT (Dispatch / Accounting / Mobile / Payroll).

## Scope

Applies to personnel responsible for:
- Onboarding drivers
- System administration
- Dispatch setup
- Accounting configuration

## Procedure

### Important Data Entry Rule

- For new entries: enter data and hit TAB to add more details
- For existing entries: type first five characters → hit Enter → select from dropdown

---

### Step 1: Add Driver as User

1. Navigate to: **System → Users**
2. Enter driver's full first and last name
3. Hit TAB to populate details
4. Save

### Step 2: Add Mobile Device

1. Navigate to: **System → Mobile → Devices**
2. Enter driver's first and last name → hit TAB
3. Device Type: Select appropriate type
4. Owned By: Personal
5. Have driver open Samsara app → About → obtain serial number
6. Enter serial number
7. Click Save and Close

### Step 3: Dispatch Setup

Navigate to: **Dispatch → Main Tab**

**Basic Info Section**
- Assign Truck (Owner Operators only)
- Enter Experience (if required)
- Set Domicile Location (select Base)

**License Section**
- Enter DL number and details

**Recruiting Section**
- Enter Hire Date

**Mobile Dispatch Section**
- Login: Start typing driver name → hit Enter → select from dropdown
- Dispatchers: Select Dispatchers
- Scans: Select Scans
- Check These Boxes: ✔ Display Financial Data ✔ Show Miles ✔ Allow Logout
- Device: Start typing driver name → select from dropdown
- Mechanics: Select Dispatchers
- General Chat: Select Dispatchers
- DO NOT SELECT: ✖ Inspect Trailers ✖ Allow to Reset Trip Data

**Ruleset**
- Add: USA Property 70/8
- Check: ✔ Short Haul Exception (if applicable)

### Step 4: Accounting Tab Setup

- Leave Accounting ID as is
- **Driver Type:**
  - Owner Operators = Lease Driver
  - Company Drivers = Company Driver
- **Pay Type:** Settlement
- **Pay Company:** Main
- **Email Settlement:**
  - Check for Company Drivers
  - DO NOT check for Owner Operator drivers
- **Leave Blank:** Bank Info, Factoring Info, Empty Pay Plan, Gate Key, SSN, Married, Sex

**Pay Plans**
- Loaded Pay Plan:
  - Owner Operators → OO 75 Percent
  - Company Drivers → CD 25 Percent
- Empty Pay Plan: Leave blank

**Additional Required Entries**
- Enter Birthdate
- Enter assigned EFS Fuel Card Number

### Step 5: Save

- Click Save and Close (top right)

---

## Part 2: Adding a New Driver in Samsara

### Step 1: Access Driver Management

1. Log in to Samsara Dashboard
2. From left-hand menu, click **Settings**
3. Select **Drivers**

### Step 2: Add Driver

1. Click **Add Driver**
2. Select Add Manually (if prompted)

### Step 3: Enter Driver Personal Information

Enter the following:
- First Name
- Last Name
- Email Address (login + notifications)
- Phone Number (if required)

### Step 4: Login & Identification

- Create/Assign Username
- Assign Driver ID (must follow company naming standard)
- Set or generate Temporary Password

### Step 5: Compliance & License Information

Enter the following:
- Driver's License Number
- License State
- License Expiration Date
- DOT or compliance details (if required)

### Step 6: Driver App & ELD Settings

- Enable Samsara Driver App Access
- Assign ELD status: Regulated or Exempt
- Set correct HOS ruleset (based on role and region)
- Assign driver to correct carrier/fleet

### Step 7: Optional Assignments

If applicable:
- Assign default vehicle
- Assign tags/groups (terminal, region, team)

### Step 8: Final Review

Verify the following:
- Name spelling
- Email/login
- License details
- HOS ruleset
- Required fields complete

### Step 9: Save & Activate

1. Click Save/Create Driver
2. Confirm driver appears in Drivers list
3. Send login details if not sent automatically

## Final Verification Checklist

Before completing onboarding, confirm:
- ✔ Driver visible in Samsara
- ✔ Driver visible in DITAT
- ✔ Correct HOS ruleset assigned
- ✔ Mobile device linked
- ✔ Pay plan correct
- ✔ Driver type correct
- ✔ Fuel card assigned
- ✔ Settlement settings correct

## Expected Outcome

- Driver compliant in Samsara
- Driver dispatch-ready in DITAT
- Payroll configured correctly
- Mobile app functional
